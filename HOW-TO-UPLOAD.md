# How to add these files (no coding — all in your web browser)

You have three files to add: **robots.txt**, **sitemap.xml**, and **vercel.json**.
They are brand-new files, so they can't break anything that already works. You'll add them straight through the GitHub website — no software to install, no command line.

## Steps

1. Go to **github.com** and open the repository where your Net Zero Search website lives (the one you pushed your HTML to).
2. Make sure you're looking at the **main folder** of the repo (the first screen you see when you open it — the list of your website files).
3. Click the **"Add file"** button near the top right, then choose **"Upload files"**.
4. **Drag all three files** (robots.txt, sitemap.xml, vercel.json) from your computer into the box on the page. *(If drag-and-drop is awkward, click "choose your files" and select them.)*
5. Scroll down. In the box that says "Commit changes," you can leave the default message or type something like `Add SEO files`. Leave "Commit directly to the main branch" selected.
6. Click the green **"Commit changes"** button.

That's it. Vercel is already connected to this repo, so it will automatically rebuild and publish your site within a minute or two. You don't need to touch Vercel at all.

## How to check it worked

After a couple of minutes, open these two links in your browser:

- https://www.netzero-search.com/robots.txt
- https://www.netzero-search.com/sitemap.xml

If you see the text of the files (not an error page), it worked.

## One thing to double-check

The **sitemap.xml** lists your main pages. I used standard page names (/about, /services, /sectors, /insights, /contact). If any of your pages have a different web address, just tell me the correct ones and I'll fix the file — it takes two seconds.
